# 学習ログ 2026/05/09

## 知らなかった概念

### 機械学習の流れ
- EDA（探索的データ分析）→ 前処理 → モデル作成 → 提出 → 改善のループ
- 最初はシンプルなモデルで動くものを作ることが大事
- 改善は1つずつ試して変化を記録する

### 評価指標
- Accuracy：正解した割合（高いほど良い）
- RMSE：予測値と正解値のズレ（低いほど良い）
- RMSE = sqrt(MSE)、二乗して平均して平方根を取ることで元の単位に戻す

### 交差検証（Cross Validation）
- データをcv個に分割して、テストするfoldを変えながら複数回評価する
- cv=5が一般的（安定性とコストのバランス）
- foldごとにスコアがバラつく場合はデータの難易度が均一でない可能性がある

### 欠損値の補完方法
- 中央値補完：シンプルだが雑。外れ値に強い
- 線形補間：前後の値を直線で結ぶ。時系列データに有効
- 平均と線形補間の違い：平均は全部同じ値、線形補間は滑らかに変化する

## 知らなかったコード

### glob
- ワイルドカードでファイルを一括取得できる
- glob.glob('path/*__horizontal_well.csv')

### groupby + transform + lambda
- 井戸ごとにグループ化して同じ処理を適用する
- transformは元と同じ行数で返すのでDataFrameに上書きできる
- applyは集計結果を返すので行数が減る

### isnull() / notnull()
- isnull()：欠損ならTrue、値があるならFalse
- notnull()：値があるならTrue（逆バージョン）
- .sum()と組み合わせて欠損数を数える

### interpolate(method='linear')
- 線形補間を実行するメソッド
- 先頭・末尾のNaNは埋められない（前後に値が必要）

### pd.concat
- 複数のDataFrameを縦に結合する
- ignore_index=Trueで行番号を0から振り直す
